# Expense Tracker

A command-line expense tracking app built with Python.

## Features
- Add expenses with item name and amount
- Saves expenses to a JSON file automatically
- Data persists between sessions
- Handles invalid input without crashing

## How to Run
1. Make sure Python is installed
2. Run: python my.py
3. Follow the prompts to add expenses

## Built With
- Python 3
- JSON for data storage
